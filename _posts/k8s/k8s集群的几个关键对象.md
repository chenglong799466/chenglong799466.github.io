

API Server、Scheduler、Controller Manager、etcd、kubelet 和 kube-proxy 组件。

API Server

API Server 是 Kubernetes 集群的入口，它提供了 Kubernetes 的 REST API 接口。

Scheduler

Scheduler 是 Kubernetes 集群的调度器，它负责将 Pod 调度到集群中的节点上。

Controller Manager

Controller Manager 是 Kubernetes 集群的控制器管理器，它负责管理 Kubernetes 集群中的各种控制器，例如 Replication Controller、Endpoint Controller、Service Controller 等。

etcd

etcd 是 Kubernetes 集群的分布式键值存储，它用于存储 Kubernetes 集群中的各种配置信息，例如 Pod、Service、Replication Controller 等。

kubelet

kubelet 是 Kubernetes 集群中的节点代理，它负责管理节点上的各种资源，例如 Pod、Container、Volume 等。
