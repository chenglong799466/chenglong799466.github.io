# 更新日志

## [2.0.0] - 2026-01-30

### 🎉 重大更新 - 极简高级风重构

#### ✨ 新增功能
- 全新的极简高级风设计风格
- 三栏布局（顶部导航 + 左侧边栏 + 主内容区 + 底部页脚）
- 归档页面（按年份时间线展示）
- 完整的响应式设计（桌面/平板/移动端）
- 统一的设计系统（配色/字体/间距/圆角/阴影）

#### 🎨 设计优化
- 采用低饱和蓝色主色调（#165DFF）
- 纯白背景，留白充足
- 清晰的字号层级（H1:32px, H2:24px, H3:20px, H4:18px）
- 使用 Inter + Noto Sans SC 字体
- 统一的 8px 基准间距系统
- 三级阴影系统
- 流畅的过渡动画（0.2s ease）

#### 📄 页面重构
- **首页**：文章卡片列表，支持分页
- **文章详情页**：优化排版，代码高亮，标签展示
- **关于页**：个人信息，技术栈，研究方向
- **标签页**：标签云，按标签分类
- **归档页**：时间线展示，统计信息
- **404页面**：友好的错误提示

#### 🔧 技术改进
- 使用 CSS Variables 管理配色
- 模块化的样式组织
- 语义化的 HTML 结构
- 优化的 CSS 选择器
- 最小化的 DOM 结构

#### 📱 响应式优化
- 桌面端（≥1024px）：三栏布局
- 平板端（768-1023px）：两栏布局
- 移动端（<768px）：单栏布局
- 优化触摸交互
- 适配小屏幕显示

#### 📚 文档完善
- 详细的 README.md
- 快速开始指南（QUICK_START.md）
- 设计规范指南（DESIGN_GUIDE.md）

- 更新日志（CHANGELOG.md）

#### 🎯 配置更新
- 更新博客标题和描述

- 优化友情链接

### 🗑️ 移除内容
